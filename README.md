# Password Manager using Python

A menu-driven Password Manager built using Python and Object-Oriented Programming.

## Features

* Add Account
* View Accounts
* Search Password
* Update Password
* Delete Account
* Password Strength Checker
* Automatic Password Generator
* Save Data using JSON
* Load Data automatically when the program starts

## Technologies Used

* Python
* OOP
* JSON
* File Handling
* Exception Handling

## Concepts Covered

* Classes and Objects
* Methods and self
* Lists and Dictionaries
* Loops and Conditions
* Random Module
* String Module
* JSON Module
* CRUD Operations

## Future Improvements

* Hide password using getpass
* Encrypt passwords
* GUI using Tkinter
* Database support
